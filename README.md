# CGPA Calculator in C

This project is a simple CGPA calculator developed in C.

## Features
- Takes input for number of courses
- Accepts grade and credit hours for each course
- Calculates total credits and grade points
- Displays the final CGPA

## Concepts Used
- Loops
- User Input
- Arithmetic Calculations

## Example Output

Enter number of courses: 3

Course 1  
Grade: 3.5  
Credit Hours: 3  

Course 2  
Grade: 3.0  
Credit Hours: 3  

Course 3  
Grade: 4.0  
Credit Hours: 2  

Final CGPA: 3.44
